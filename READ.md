# Network Analyzer

Script Python che analizza la rete locale e genera un report completo.

## Cosa fa
- Mostra IP e interfacce di rete
- Mostra il gateway
- Mostra le porte aperte sul sistema
- Esegue una query DNS
- Salva tutto in un report con timestamp

## Tecnologie usate
- Python 3
- subprocess
- Linux (Debian)

## Come si usa
1. Esegui con `python3 network_analyzer.py`
2. Il report viene stampato a schermo e salvato in `network_report.txt`
